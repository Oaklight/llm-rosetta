# 快速开始

欢迎使用 LLMIR！本指南将帮助您快速上手，了解如何安装和使用 LLMIR 进行 LLM 消息格式转换。

## 概述

LLMIR (Large Language Model Intermediate Representation) 是一个专门用于处理不同 LLM 提供商消息格式转换的 Python 库。它提供了统一的中间表示，让您可以轻松地在不同的 LLM 提供商之间转换消息格式。

## 主要功能

- **统一的消息格式转换**：支持 OpenAI、Anthropic、Google 等主流提供商
- **自动格式检测**：智能识别输入消息的格式类型
- **类型安全**：完整的类型注解，提供良好的开发体验
- **高性能**：优化的转换算法，最小化性能开销

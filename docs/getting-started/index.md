# Getting Started

Welcome to LLMIR! This guide will help you get up and running quickly with LLMIR for LLM message format conversion.

## Overview

LLMIR (Large Language Model Intermediate Representation) is a Python library specifically designed for handling message format conversion between different LLM providers. It provides a unified intermediate representation that allows you to easily convert message formats between different LLM providers.

## Key Features

- **Unified message format conversion**: Support for OpenAI, Anthropic, Google, and other major providers
- **Automatic format detection**: Intelligent identification of input message format types
- **Type safety**: Complete type annotations for excellent development experience
- **High performance**: Optimized conversion algorithms with minimal performance overhead

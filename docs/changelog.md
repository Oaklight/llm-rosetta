## 版本说明

我们遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范。

### 版本格式

- **主版本号**：当您做了不兼容的 API 修改
- **次版本号**：当您做了向下兼容的功能性新增
- **修订号**：当您做了向下兼容的问题修正

### 变更类型

- **新增** - 新功能
- **变更** - 对现有功能的变更
- **弃用** - 即将移除的功能
- **移除** - 已移除的功能
- **修复** - 问题修复
- **安全** - 安全相关的修复
